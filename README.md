# Grama Kalyana Sports

Grama Kalyana Sports is an Android application developed using Kotlin and Firebase to manage village-level sports tournaments and community sports events. The app helps organizers and players manage matches, track scores, view leaderboards, and monitor tournament activities in real time.

---

## Features

* Live score updates
* Match scheduling and management
* Player and team management
* Tournament tracking
* Leaderboard system
* Firebase Realtime Database integration
* Modern Jetpack Compose UI
* Real-time data synchronization

---

## Technologies Used

* Kotlin
* Jetpack Compose
* Firebase Realtime Database
* Firebase Authentication
* Android Studio
* Material Design
* Gradle
* JSON

---

## Problem Statement

Managing local sports tournaments manually is difficult and time-consuming. Organizers often face challenges in updating match scores, tracking player performance, and sharing tournament information efficiently.

Grama Kalyana Sports solves this problem by providing a digital platform for tournament management, live score updates, leaderboard tracking, and player management.

## Installation Steps

1. Clone the repository:

```bash
https://github.com/pavankumarvn1818/Grama-Kalyana-Sports.git
```

2. Open the project in Android Studio.

3. Sync Gradle files.

4. Add your Firebase configuration file (`google-services.json`) inside the `app/` folder.

5. Run the application on an emulator or Android device.
